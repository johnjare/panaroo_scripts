# pop_genes.py
# Source: https://github.com/johnjare
# Created by Jared Johnson
#
# Generates lists of genes unique to each input population 
# based on the supplied gene presence-absence matrix.
# ---------------------------------------------------------
# Example:
# 
# > python3 pop_genes.py -i gene_presence_absence.Rtab -p1 pop1.lst -p2 pop2.lst -o1 p1_genes.lst -o2 p2_genes.lst -a True
# 
# Inputs:
# gene_presence_absence.Rtab - presence/absence matrix generated by Roary or Panaroo
# pop1.lst - list of isolates present in population 1
# pop2.lst - list of isolates present in population 2
#
# Outputs:
# p1_genes.lst - all genes unique to population 1
# p2_genes.lst - all genes unique to population 2
# accessory_genes.lst - a list of accessory genes (i.e., those genes not present in all isolates)
