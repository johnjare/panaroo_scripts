# accessory_binary_genes.py
# Source: https://github.com/johnjare
# Created by Jared Johnson
#
# Reformats a gene presence/absence matrix generated from Panaroo or Roary into a FASTA-like format. The output can be used to generate a phylogenetic tree.
# ---------------------------------------------------------
# Example:
# 
# > python3 accessory_binary_genes.py -i gene_presence_absence.Rtab -o accessory_binary_genes.binary
# 
# Inputs:
# gene_presence_absence.Rtab - presence/absence matrix generated by Roary or Panaroo
#
# Outputs:
# accessory_binary_genes.binary - binary alignment file of accessory genes for all isolates in the gene presence/absence matrix.
